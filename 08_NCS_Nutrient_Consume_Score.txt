================================================================================
        NCS (Nutrient Consume Score) — 营养消费评分系统详解
================================================================================
论文(预印本)：A Nutrient Ratio-Based, Web-Enabled Food Quality Score Performs
              Favorably Compared with Leading Nutrient Profiling Systems
              for Weight and Blood Pressure
期刊：medRxiv (2024.12.23), DOI: 10.1101/2024.12.23.24319328
作者：Damman CJ, Frankenfeld CL
机构：University of Washington（华盛顿大学）
数据：USDA FoodData Central (FNDDS 2021-2023)，NHANES 2015-2016成人>=20岁
人群：美国人群（预印本阶段，验证数据来自美国NHANES）
适用范围：设计为消费者手机端(web-enabled)即时评分系统

================================================================================
评分体系总览
================================================================================
评分范围：1-100 分
三个颜色区间：
  >= 70 → 绿色「鼓励多食」
  60-69 → 黄色「适量食用」
  < 60  → 红色「尽量少食」

核心理念：以「营养比率」代替「绝对值」来捕捉未加工全食品中的自然平衡
          食品加工会破坏这些比率——浓缩糖/饱脂/钠，去除纤维/不饱和脂肪/钾

================================================================================
三大核心营养比率（子分数）
================================================================================

┌────────────────────────┬─────────────────────┬────────────────────────────┐
│ 比率（子分数）         │ 计算公式            │ 健康原理                   │
├────────────────────────┼─────────────────────┼────────────────────────────┤
│ 1. 碳纤比率            │ 总碳水(g)           │ 纤维减缓糖吸收，被发酵为    │
│    (Carb Quality)      │ / 膳食纤维(g)       │ 丁酸(调节食欲和血糖)。       │
│                        │                     │ 比率越低越好               │
├────────────────────────┼─────────────────────┼────────────────────────────┤
│ 2. 饱脂/不饱脂比率     │ 饱和脂肪(g)         │ 反映细胞膜脂肪组成，        │
│    (Fat Quality)       │ / 不饱和脂肪(g)     │ 饱脂主导→炎症和体重增加。   │
│                        │                     │ 比率越低越好               │
├────────────────────────┼─────────────────────┼────────────────────────────┤
│ 3. 钠钾比              │ 钠(mg) / 钾(mg)     │ 模拟细胞膜离子泵机制，      │
│    (Salt Quality)      │                     │ 影响血压和微生物组健康。     │
│                        │                     │ 比率越低越好               │
└────────────────────────┴─────────────────────┴────────────────────────────┘

每个比率产生一个子分数。比率<2通常与较健康的未加工/包装食品相关。

================================================================================
评分算法（已知部分）
================================================================================

Step 1：从USDA FNDDS提取每100g或每份食物的营养素
Step 2：计算三个比率：
  R1 = 总碳水(g) / 膳食纤维(g)
  R2 = 饱和脂肪(g) / 不饱和脂肪(g)
  R3 = 钠(mg) / 钾(mg)

Step 3：将每个比率转换为子分数（连续尺度，比率越低子分越高）
  （注：精确的转换函数、阈值和加权在公开材料中未完全披露）

Step 4：生物活性化合物调整（基于食品类别代理）
  因多酚/omega-3/可发酵纤维等通常不在营养标签上，
  NCS使用USDA食品类别（水果/蔬菜/全谷物/豆类/坚果/种子等）作为代理
  为未加工全食品类别加分

Step 5：酒精调整
  明确纳入酒精的热量和健康影响——其他大多数NPS忽略此项

Step 6：合并子分数+调整 → 最终1-100 NCS分数

================================================================================
研究结果 (medRxiv 2024)
================================================================================
- 所有五种NPS(NCS/NS/HS/FC2/NOVA)均与更健康的体重指标显著相关
- NCS和Food Compass 2.0在BMI和腰围方面显示出最大的beta系数
  和最低的肥胖优势比
- NCS(不含酒精)和Nutri-Score在收缩压和舒张压方面关联最强
- 纳入酒精和生物活性调整改善了NCS与体重结局的关联
- 食品类别间的高分/低分食物在所有5个NPS中基本一致

================================================================================
实际应用
================================================================================
- 已有免费beta版手机App（条形码扫描获取NCS分数和各比率子分）
- 子分数也可用于指导食物搭配（如高碳/低纤食物配高纤食物优化整体比率）
- 使用美国营养标签上已有的数据，不需要额外的数据库

================================================================================
对植物基替代品的立场
================================================================================
NCS的一个独特立场：因使用营养比率而非加工程度，植物奶等替代品可以获得较高
分数（如果其碳纤比、脂质比、钠钾比合理），不会像NOVA那样将所有含添加剂的
食品一律归为最低级。这是对现有系统"过度惩罚加工食品"的一个制衡观点。
