================================================================================
        DQI-I (Diet Quality Index-International) — 国际饮食质量指数详解
================================================================================
论文：The Diet Quality Index-International (DQI-I) Provides an Effective Tool
      for Cross-National Comparison of Diet Quality as Illustrated by China
      and the United States
期刊：Journal of Nutrition (2003), Vol. 133, pp. 3476-3484
作者：Kim S, Haines PS, Siega-Riz AM, Popkin BM
机构：北卡罗来纳大学教堂山分校
数据：中国(CHNS)和美国(CSFII)全国膳食数据
人群：跨国对比（首次设计可跨文化跨国比较的饮食质量指数）
适用范围：全球通用，已在中国、美国、墨西哥、西班牙、地中海等地验证

================================================================================
评分体系总览
================================================================================
总评分范围：0-100 分
  >= 60% (>=60分) → 可接受的饮食质量
  < 60% (<60分) → 低质量饮食（"poor-quality diet"）

四个大类维度，涵盖食物层面和营养素层面：

┌────────────────────┬──────────┬──────────────────────────────────┐
│ 维度                │ 分值范围 │ 组件数                           │
├────────────────────┼──────────┼──────────────────────────────────┤
│ 1. 多样性 Variety   │ 0-20     │ 2个子组件                        │
│ 2. 充足度 Adequacy  │ 0-40     │ 8个组件(每组件0-5分连续)         │
│ 3. 适度 Moderation  │ 0-30     │ 5个组件(每组件0/3/6分三级)       │
│ 4. 总体平衡 Balance │ 0-10     │ 2个子组件                        │
│ TOTAL               │ 0-100    │ 17个组件                         │
└────────────────────┴──────────┴──────────────────────────────────┘

================================================================================
一、多样性 (Variety, 0-20分)
================================================================================

1A. 五大食物组多样性 (0-15分)
  评估是否每日摄入来自5大食物组的至少1份：
    1. 肉类/禽肉/鱼/蛋
    2. 乳制品/豆类（合并，因发展中国家乳制品摄入低、豆类是主要钙来源）
    3. 谷物
    4. 水果
    5. 蔬菜

┌──────────┬──────────────────────────────────────────────────┐
│ 分数     │ 标准                                              │
├──────────┼──────────────────────────────────────────────────┤
│ 15       │ 每日来自所有5组的>=1份                            │
│ 12       │ 缺少任意1组                                       │
│  9       │ 缺少任意2组                                       │
│  6       │ 缺少任意3组                                       │
│  3       │ 缺少>=4组                                         │
│  0       │ 所有组均无                                        │
└──────────┴──────────────────────────────────────────────────┘

1B. 蛋白质来源组内多样性 (0-5分)
  评估6种蛋白来源(肉/禽/鱼/乳制品/豆类/蛋)中的摄入种类数
  >0.5份/天计为有意义的摄入

┌──────────┬──────────────────────────────────────────┐
│ 分数     │ 标准                                      │
├──────────┼──────────────────────────────────────────┤
│  5       │ >=3种不同来源/天                          │
│  3       │ 2种不同来源/天                            │
│  1       │ 1种来源/天                                │
│  0       │ 无                                        │
└──────────┴──────────────────────────────────────────┘

================================================================================
二、充足度 (Adequacy, 0-40分)
================================================================================
8个组件，每组件0-5分连续计分（与推荐摄入量达成比例，封顶5分）
使用各国家自身的DRI/RDA/RNI标准（不使用统一标准）

┌───┬──────────────┬────────────────────────────────┬──────────────────┐
│ # │ 组件         │ 计分基础                        │ 满分(5分)标准    │
├───┼──────────────┼────────────────────────────────┼──────────────────┤
│ 1 │ 蔬菜组       │ 份/天(能量依赖)                 │ >=3-5份/天       │
│ 2 │ 水果组       │ 份/天(能量依赖)                 │ >=2-4份/天       │
│ 3 │ 谷物组       │ 份/天(能量依赖)                 │ >=6-11份/天      │
│ 4 │ 膳食纤维     │ g/天(能量依赖)                  │ >=20-30g/天      │
│ 5 │ 蛋白质       │ %能量                           │ >=10%总能量/天   │
│ 6 │ 铁           │ % RDA(或AI)                     │ >=100% RDA/AI    │
│ 7 │ 钙           │ % AI                            │ >=100% AI        │
│ 8 │ 维生素C      │ % RDA(或RNI)                    │ >=100% RDA/RNI   │
└───┴──────────────┴────────────────────────────────┴──────────────────┘

能量依赖的份量阈值（三个热量水平）：

┌──────────┬──────────┬──────────┬──────────┐
│ 组件     │ 1700kcal │ 2200kcal │ 2700kcal │
├──────────┼──────────┼──────────┼──────────┤
│ 蔬菜     │ >=3份    │ >=4份    │ >=5份    │
│ 水果     │ >=2份    │ >=3份    │ >=4份    │
│ 谷物     │ >=6份    │ >=9份    │ >=11份   │
│ 纤维     │ >=20g    │ >=25g    │ >=30g    │
└──────────┴──────────┴──────────┴──────────┘

================================================================================
三、适度 (Moderation, 0-30分)
================================================================================
5个组件，三级计分(0/3/6)，关注慢性病相关的限制性摄入

┌───┬──────────────────┬──────────┬──────────────┬──────────┐
│ # │ 组件             │ 6分(最优)│ 3分(中等)    │ 0分(最差)│
├───┼──────────────────┼──────────┼──────────────┼──────────┤
│ 1 │ 总脂肪           │ <=20%能量│ >20-30%能量  │ >30%能量 │
│ 2 │ 饱和脂肪         │ <=7%能量 │ >7-10%能量   │ >10%能量 │
│ 3 │ 胆固醇           │ <=300mg/d│ >300-400mg/d │ >400mg/d │
│ 4 │ 钠               │ <=2400mg │ >2400-3400mg │ >3400mg  │
│ 5 │ 空卡路里食品     │ <=3%能量 │ >3-10%能量   │ >10%能量 │
└───┴──────────────────┴──────────┴──────────────┴──────────┘

"空卡路里食品"定义（DQI-I独特创新）：
  指提供能量但营养素密度不足的食物（如食糖、油、酒精）
  营养素密度 = (食物中某营养素量 / 该营养素推荐摄入量)
            / (食物能量 / 推荐能量摄入量)
  当所有评估营养素密度之和 < 1 时 → 定义为空卡路里
  原论文美国用15种营养素，中国替换了部分（因数据可用性）

================================================================================
四、总体平衡 (Overall Balance, 0-10分)
================================================================================

4A. 宏量营养素比例 (0-6分)
  碳水:蛋白:脂肪的能量比

┌──────────┬──────────────────────────────────────────┐
│ 分数     │ 标准(C:P:F)                               │
├──────────┼──────────────────────────────────────────┤
│  6       │ 55-65 : 10-15 : 15-25                    │
│  4       │ 52-68 : 9-16 : 13-27                     │
│  2       │ 50-70 : 8-17 : 12-30                     │
│  0       │ 以上范围之外                              │
└──────────┴──────────────────────────────────────────┘

4B. 脂肪酸比例 (0-4分)
  P/S = PUFA/SFA; M/S = MUFA/SFA

┌──────────┬──────────────────────────────────────────┐
│ 分数     │ 标准                                      │
├──────────┼──────────────────────────────────────────┤
│  4       │ P/S=1-1.5 且 M/S=1-1.5                   │
│  2       │ P/S=0.8-1.7 且 M/S=0.8-1.7              │
│  0       │ 以上范围之外                              │
└──────────┴──────────────────────────────────────────┘

================================================================================
原始应用结果 (中国 vs 美国, Kim 2003)
================================================================================
┌────────────────────┬──────────┬──────────┐
│ 维度                │ 中国     │ 美国     │
├────────────────────┼──────────┼──────────┤
│ 多样性 (max 20)     │ 11.8     │ 15.6     │
│ 充足度 (max 40)     │ 28.0     │ 28.1     │
│ 适度 (max 30)       │ 18.6     │ 14.3     │
│ 总体平衡 (max 10)   │  2.1     │  1.1     │
│ 总分 (max 100)      │ 60.5     │ 59.1     │
└────────────────────┴──────────┴──────────┘

中国优势：更优的适度得分（总脂肪低）、稍好平衡
美国优势：更多样化的食物选择
共同问题：两国宏量营养素/脂肪酸比例均不平衡

================================================================================
核心设计特点
================================================================================
1. 双重方法：同时评估食物层面和营养素层面
2. 国家特异性DRIs：各国用自身RDA/RNI/AI标准（考虑到铁生物利用度等差异）
3. 份量标准化：使用美国食物指南金字塔的份量定义（跨国对比统一用）
4. 能量调整：蔬菜/水果/谷物/纤维的推荐量按3个能量水平缩放
5. 中国人的DQI-I使用注意事项：
   - 纤维在中国可能被低估（中国食物成分表限制）
   - 中国铁RDA低于美国(因铁生物利用度差异)
   - 不适合未修正的地中海饮食（橄榄油的高总脂肪会被惩罚）

================================================================================
已知局限性
================================================================================
- 不适用于地中海饮食（高橄榄油→高总脂肪→适度维度低分）
  已有地中海DQI-I改编版(Mariscal-Arcas et al., 2007)将橄榄油与饱和脂肪区分
- 短期回忆(2-3天24h)可能错误分类多样性
- 原版未包含反式脂肪酸
- 份量标准化依赖美国的定义
