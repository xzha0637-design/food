================================================================================
        NOVA — 食品加工程度四级分类系统详解
================================================================================
提出：Carlos Monteiro 教授，巴西圣保罗大学，2009年首次提出，2014年定型
性质：按「加工程度和目的」分类，不按营养素分类，天然跨文化
人群：全球通用（FAO已在其全球膳食指南中采用）
适用范围：所有食品和饮料

================================================================================
NOVA四级分类
================================================================================

┌──────────────────────────────────────────────────────────────────────┐
│ NOVA 第1组：未加工或最低加工食品                                        │
├──────────────────────────────────────────────────────────────────────┤
│ 定义：食用植物/动物/菌类/藻类的可食部分，分离自自然                     │
│ 允许的加工：去除不可食用部分、干燥、粉碎、研磨、分馏、烘焙、煮沸、      │
│            巴氏消毒、冷藏、冷冻、真空包装、非酒精发酵                   │
│ 核心规则：不添加盐/糖/油或其他任何物质到原始食物中                       │
│ 目的：延长保质期、便于储存和制备                                        │
│                                                                      │
│ 示例：                                                                 │
│  · 新鲜/冷冻水果蔬菜                                                   │
│  · 谷物(糙米/白米/玉米/小麦粒)                                         │
│  · 玉米/小麦/燕麦面粉                                                   │
│  · 生或烤制坚果/种子(无添加盐/糖)                                       │
│  · 豆类(豆/扁豆/鹰嘴豆)                                                 │
│  · 土豆、红薯                                                           │
│  · 新鲜/冷藏/冷冻肉类、禽肉、鱼、海鲜                                   │
│  · 鸡蛋                                                                 │
│  · 新鲜或巴氏消毒纯牛奶                                                 │
│  · 原味酸奶(无添加糖/调味)                                             │
│  · 现榨果蔬汁(无添加糖)                                                 │
│  · 香草和香料                                                           │
│  · 茶、咖啡、水                                                         │
│  · 仅面粉+水制作的意面                                                  │
│  · 无添加糖/蜂蜜/油的谷物+坚果+干果麦片                                │
└──────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────┐
│ NOVA 第2组：加工烹饪配料                                                │
├──────────────────────────────────────────────────────────────────────┤
│ 定义：从第1组食物或自然中直接提取的物质                                  │
│ 允许的加工：压榨、离心、精炼、提取、采矿、研磨、干燥                    │
│ 核心特征：不单独食用，用于在第1组食物基础上的家庭/餐厅烹饪               │
│                                                                      │
│ 示例：                                                                 │
│  · 植物油(橄榄油/椰子油等)                                             │
│  · 黄油和猪油                                                           │
│  · 食糖和糖蜜(甘蔗/甜菜)                                               │
│  · 蜂蜜                                                                 │
│  · 枫糖浆                                                               │
│  · 醋                                                                   │
│  · 从玉米/植物提取的淀粉                                               │
│  · 盐(开采或海盐)、碘盐                                                │
│  · 加盐黄油（两个第2组配料组合）                                       │
│ 目的：使第1组食物的多样化制备成为可能                                   │
└──────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────┐
│ NOVA 第3组：加工食品                                                     │
├──────────────────────────────────────────────────────────────────────┤
│ 定义：第1组食物 + 至少一种第2组配料(盐/糖/油)制成的简单改良食品          │
│ 核心规则：仅由第1组+第2组配料组成，不添加任何化妆品类添加剂             │
│ 允许的加工：罐装、瓶装、腌制、烟熏、盐腌、发酵、煮沸、烘焙、非酒精发酵 │
│                                                                      │
│ 示例：                                                                 │
│  · 盐水罐装蔬菜/豆类                                                   │
│  · 糖渍水果                                                             │
│  · 盐渍/糖渍坚果和种子                                                 │
│  · 盐腌/风干/罐装肉类/禽肉/鱼                                          │
│  · 奶酪                                                                 │
│  · 新鲜制作(手工)面包                                                   │
│  · 啤酒、葡萄酒、苹果酒                                                 │
│ 目的：增加第1组食物的耐久性和改善感官品质                               │
└──────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────┐
│ NOVA 第4组：超加工食品 (UPF) ★核心关注组★                              │
├──────────────────────────────────────────────────────────────────────┤
│ 定义：工业配方，主要/全部由从食物中提取的物质、化学改性的食物成分、      │
│      化妆品添加剂组成，几乎不含完整食物                                  │
│ 典型成分数：5种或更多                                                    │
│                                                                      │
│ 超加工标志物类别一（非厨房成分）：                                      │
│  · 改性糖：果糖、高果糖玉米糖浆、浓缩果汁、转化糖、麦芽糊精             │
│  · 改性油：氢化油、酯交换油                                             │
│  · 改性蛋白：水解蛋白、大豆分离蛋白、谷朊粉、酪蛋白、乳清蛋白、         │
│    机械分离肉                                                           │
│  · 改性淀粉                                                             │
│                                                                      │
│ 超加工标志物类别二（化妆品添加剂）：                                    │
│  · 香料、增味剂(味精)、色素、甜味剂(含非糖甜味剂)                      │
│  · 乳化剂、乳化盐                                                       │
│  · 增稠剂、抗泡剂、填充剂、碳酸化剂、发泡剂、凝胶剂、抛光剂             │
│                                                                      │
│ 超加工食品示例：                                                       │
│  · 碳酸软饮料、果汁饮料、能量饮料                                       │
│  · 加糖/调味酸奶                                                        │
│  · 糖果、人造黄油                                                       │
│  · 鸡块/鱼条、香肠、热狗、午餐肉、所有重组肉制品、植物肉替代品          │
│  · 挤压早餐麦片                                                         │
│  · 速溶汤/面/甜点                                                       │
│  · 婴儿配方奶和后续奶                                                   │
│  · 代餐奶昔和粉末                                                       │
│  · 大规模生产的包装面包(含乳化剂、添加谷朊等)——手工面包为第3组         │
│  · 冰淇淋、包装零食(甜/咸)、冷冻即食餐、预制派/披萨                     │
│                                                                      │
│ 识别规则（Monteiro 2019）：                                            │
│ "实用识别法：查看成分清单中是否含至少一种NOVA第4组特征成分"              │
│ 已编目了超过60种超加工标志添加剂                                        │
│                                                                      │
│ 目的：创造高利润(低成本原料+长保质期+品牌化)、极便利(即食)、            │
│      超可口的产品，易取代所有其他NOVA组食物                             │
└──────────────────────────────────────────────────────────────────────┘

================================================================================
核心度量指标
================================================================================
超加工食品的膳食占比（Dietary Share of UPF）：
  = 来自所有NOVA第4组产品的总膳食摄入量(能量或重量)百分比

应用方法：
  1. 收集膳食数据(24小时回顾/食物频率问卷)
  2. 将每种食物编码到NOVA组
  3. 计算第4组食品占总能量摄入的百分比

在全球层面：
  美国：~58%的卡路里来自UPF
  英国：~57%的卡路里来自UPF

================================================================================
2024年关键进展与争议
================================================================================

1. BMJ伞状综述 (2024)：
   Lane MM等，45项研究，近1000万人
   UPF暴露与32种健康参数直接相关（死亡/癌症/心理健康/呼吸/心血管等）
   但大多数结局的证据质量评为低或极低

2. Diabetes Care研究 (2024)：
   某些UPF子类(早餐麦片/全麦面包/酸奶/乳制品)降低2型糖尿病风险
   引发对NOVA分类精度的质疑

3. USDA资助研究 (J. Nutrition)：
   用>80% NOVA第4组食品构建2000kcal膳食
   获得HEI评分86/100(vs美国当前平均59)
   证明UPF原则上可以是高质量膳食的一部分

4. 主要批评：
   - 不精确：手工法棍=第3组；含维生素C的超市面包=第4组
   - 误分类：含瓜尔胶的鹰嘴豆泥=第4组，与苏打水和糖果同级别
   - "搭便车"问题：许多营养合理的食品仅因一两种添加剂归入第4组
   - 没有公认的全球统一定义，NOVA未被FDA/EFSA正式采用

================================================================================
与其他评分系统的关系
================================================================================
- GDQS与NOVA相关：巴西GDQS验证中，GDQS与UPF摄入呈负相关
- Food Compass 2.0将加工程度纳入为其9维度之一
- NOVA是独特的「过程导向」分类，其他系统多数为「营养素导向」
- 优势：天然跨文化，不依赖食物成分数据库
