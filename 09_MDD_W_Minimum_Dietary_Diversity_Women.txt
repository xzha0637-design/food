================================================================================
        MDD-W (Minimum Dietary Diversity for Women) — 女性最低膳食多样性指标详解
================================================================================
全称：Minimum Dietary Diversity for Women (MDD-W)
开发：FAO（联合国粮农组织）和 FHI 360/FANTA
共识时间：2014年7月技术会议
正式发布：FAO & FHI 360, 2016
前身：Women's Dietary Diversity Score (WDDS, 9食物组)
人群：非孕期育龄妇女(15-49岁)
适用范围：全球中低收入国家(LMICs)营养监测，已作为FAO/WHO全球标准

================================================================================
评分体系总览
================================================================================
类型：二分类群体指标(是/否达到最低膳食多样性)
评分范围：0-10 (食物组多样性分数, FGDS)
阈值：FGDS >= 5/10 → 达到"最低膳食多样性"（微量营养素充足的饮食）
      FGDS <  5/10  → 未达标

核心理念：10个食物组中至少消费5个 → 微量营养素充足度（验证了11种微量营养素）
          所有食物组等权（无差异加权）

================================================================================
评分方法
================================================================================

第1步：食物组多样性分数 (FGDS)
  使用24小时膳食回顾数据
  消费了某食物组 = 1分（满足15g最低摄入阈值后）
  未消费 = 0分
  FGDS范围：0（无任何食物组）到 10（全部食物组）
  所有10组等权

第2步：二分类判定
  FGDS >= 5 → "达到最低膳食多样性"(MDD-W达标)
  FGDS <  5 → "未达到最低膳食多样性"

第3步：群体指标计算
  MDD-W达标率(%) = (FGDS>=5的妇女人数) / (15-49岁受访妇女总数) × 100

================================================================================
最低摄入量阈值（15g规则）
================================================================================
- 对每个主食物组应用15g最低摄入阈值
- 同一组内多个食物合并计算（如杏仁5g + 核桃5g = 坚果和种子10g → 不计数）
- 排除调味料、调味品和微量用量（营养上无意义）
- 可选额外类别（见下）无最低量阈值

================================================================================
问卷结构（实际操作中）
================================================================================

问卷包含多于10行的问题（为便于分类和数据质量）：

拆分的子组（仍计为1个主组）：
  - 谷物/白色根茎/芭蕉 → (a)谷物及制品 (b)白色根茎/块茎和芭蕉
  - 肉/禽/鱼 → (a)肉类 (b)禽肉 (c)鱼和海鲜
  - 其他维生素A丰富蔬果 → (a)VA丰富水果 (b)VA丰富蔬菜(非叶菜)

结果：10个主组由14个问卷行覆盖

额外可选类别（不计入MDD-W分数）：
  - 昆虫和其他小型蛋白食物
  - 红棕榈油
  - 其他油脂
  - 咸味和油炸零食
  - 甜食
  - 含糖饮料

必需但不计分类别：
  - 调味品和调味料
  - 其他饮料和食品

================================================================================
数据收集方式
================================================================================

方法一：开放式回顾（推荐）
  受访者自由回忆前一日(白天+夜晚)所有饮食
  调查员随后分类到10个食物组

方法二：列表法
  调查员逐组询问"你昨天是否吃了XX组的食物？"
  记录是/否

================================================================================
与GDQS的关系
================================================================================
- MDD-W是GDQS的前身/简化版本
- MDD-W只评估10个组(等权)，GDQS评估25个组(不等权)
- MDD-W只关注微量营养素充足度(不加惩罚不健康食物)
- GDQS同时捕捉营养不良和NCD风险
- 在2024年21国对比研究中，MDD-W的跨背景预测准确性优于GDQS+
  (Sattamini et al., CDN 2025)

================================================================================
验证结果
================================================================================
- WDDP Study Group (2017, CDN)：
  FGDS >= 5的阈值在5个国家(孟加拉国/布基纳法索/马里/马拉维/莫桑比克)验证
  对11种微量营养素充足度(MMDA)的灵敏度和特异性均较好
- 后续在10个非洲国家、印度、中国、泰国、墨西哥等进一步验证
- 与24小时膳食回忆相比，MDD-W的操作负担极低，适合大规模监测

================================================================================
总结公式
================================================================================

  MDD-W达标率(%) = (∑ FGDS>=5的妇女) / (受访15-49岁妇女总数) × 100

  其中 FGDS = sum(10个二分类分数 0/1)
  每类满足15g最低摄入阈值 = 1

================================================================================
核心特点
================================================================================
1. 极简设计：仅10组，24小时，是操作最简的全球饮食质量指标之一
2. 低成本/低负担：适合低资源环境的大规模人口监测
3. 仅评估充足度(微量营养素)：不评估过度摄入和NCD风险
4. 不依赖食物成分数据库：全食物组，无营养素计算
5. 全球共识：FAO/WHO认可的全球标准
