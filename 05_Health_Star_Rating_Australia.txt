================================================================================
        Health Star Rating (HSR) — 澳大利亚/新西兰健康星级评分系统详解
================================================================================
全称：Health Star Rating System
推出：2014年（2020年修订），澳大利亚和新西兰政府联合
性质：自愿性包装正面标签，评分为0.5到5星（0.5增量）
计算基础：每100g或每100ml
底层算法：基于FSANZ Standard 1.2.7的NPSC(营养分析评分标准)修改版
适用范围：包装食品（婴儿配方、酒精、运动食品、香草/香料、茶/咖啡、单一新鲜食品除外）
人群：澳大利亚/新西兰消费者

================================================================================
六大食品类别
================================================================================
┌──────────┬──────────────────────────────────────────┐
│ 类别     │ 描述                                      │
├──────────┼──────────────────────────────────────────┤
│ Cat 1    │ 非乳饮料 (水、软饮、果汁等)               │
│ Cat 1D   │ 乳饮料 (牛奶、酸奶饮料，钙>80mg/200mL)     │
│ Cat 2    │ 非乳非饮料食品 (谷物、零食等)             │
│ Cat 2D   │ 乳类食品 (酸奶、软奶酪等，非饮料非硬奶酪) │
│ Cat 3    │ 油和涂抹酱 (黄油、人造黄油、食用油)       │
│ Cat 3D   │ 奶酪和加工奶酪 (钙>320mg/100g)            │
└──────────┴──────────────────────────────────────────┘

================================================================================
核心算法（5步计算）
================================================================================

第1步：确定食品类别
  检查是否为乳制品：含钙>320mg/100g(或>80mg/200mL) → 归入D类
  否则按饮品/油脂涂抹酱/通用食品分类

第2步：计算基线点数（负面营养素）——上限30点

┌──────────┬──────────┬──────────┬──────────┬──────────┐
│ 基线点数 │ 能量(kJ) │ 饱脂(g)  │ 总糖(g)  │ 钠(mg)   │
├──────────┼──────────┼──────────┼──────────┼──────────┤
│  0       │ <=335    │ <=1.0    │ <=5.0    │ <=90     │
│  1       │ >335     │ >1.0     │ >5.0     │ >90      │
│  2       │ >670     │ >2.0     │ >9.0     │ >180     │
│  3       │ >1005    │ >3.0     │ >13.5    │ >270     │
│  4       │ >1340    │ >4.0     │ >18.0    │ >360     │
│  5       │ >1675    │ >5.0     │ >22.5    │ >450     │
│  6       │ >2010    │ >6.0     │ >27.0    │ >540     │
│  7       │ >2345    │ >7.0     │ >31.0    │ >630     │
│  8       │ >2680    │ >8.0     │ >36.0    │ >720     │
│  9       │ >3015    │ >9.0     │ >40.0    │ >810     │
│ 10       │ >3350    │ >10.0    │ >45.0    │ >900     │
└──────────┴──────────┴──────────┴──────────┴──────────┘

基线点数 = 能量点 + 饱脂点 + 总糖点 + 钠点

第3步：计算修正点数（正面营养素）——从基线扣除

3A. V点 (FVNL = 水果/蔬菜/坚果/豆类含量)
  基于成分百分比，最多8分
  包含：新鲜、干制、冷冻、罐装、腌制、泥状或去皮的水果/蔬菜/坚果/豆类/种子/香草/香料/菌类/藻类

3B. P点 (蛋白质)——仅基线<13点时可用，与F点合计上限15分
┌──────────┬──────────────┐
│ P点数    │ 蛋白质(g)    │
├──────────┼──────────────┤
│  0       │ <=1.6        │
│  1       │ >1.6         │
│  2       │ >3.2         │
│  3       │ >4.8         │
│  4       │ >6.4         │
│  5       │ >8.0         │
└──────────┴──────────────┘

3C. F点 (膳食纤维)——Cat 1饮料不可计分
┌──────────┬──────────────┐
│ F点数    │ 纤维(g)      │
├──────────┼──────────────┤
│  0       │ <=0.9        │
│  1       │ >0.9         │
│  2       │ >1.9         │
│  3       │ >2.8         │
│  4       │ >3.7         │
│  5       │ >4.7         │
└──────────┴──────────────┘

第4步：最终分数 = 基线点数 - V点 - P点 - F点
  分数越低 = 产品越健康

第5步：最终分数转换为星级（各类别映射表不同）

┌──────┬─────────┬─────────┬─────────┬─────────┬─────────┬─────────┐
│ 星级 │ Cat 1   │ Cat 1D  │ Cat 2   │ Cat 2D  │ Cat 3   │ Cat 3D  │
│      │ 非乳饮料│ 乳饮料  │ 普通食品│ 乳食品  │ 油/涂抹 │ 奶酪    │
├──────┼─────────┼─────────┼─────────┼─────────┼─────────┼─────────┤
│ 5星  │ <=-6    │ <=-1    │ <=-11   │ <=-2    │ <=13    │ <=22    │
│ 4.5  │ -5      │ -1      │ -10--7  │ -1      │ 14-16   │ 23-24   │
│ 4    │ -4      │ 0       │ -6--2   │ 0       │ 17-20   │ 25-26   │
│ 3.5  │ -3      │ 1       │ -1 到 2 │ 1       │ 21-23   │ 27-28   │
│ 3    │ -2      │ 2       │ 3-6     │ 2       │ 24-27   │ 29-30   │
│ 2.5  │ -1      │ 3       │ 7-11    │ 3       │ 28-30   │ 31-32   │
│ 2    │ 0       │ 4       │ 12-15   │ 4       │ 31-34   │ 33-34   │
│ 1.5  │ 1       │ 5       │ 16-20   │ 6       │ 35-37   │ 35-36   │
│ 1    │ 2       │ 6       │ 21-24   │ 6       │ 38-41   │ 37-39   │
│ 0.5  │ >=3     │ >=7     │ >=25    │ >=7     │ >=42    │ >=39    │
└──────┴─────────┴─────────┴─────────┴─────────┴─────────┴─────────┘

================================================================================
计算示例：含糖软饮料 (Cat 1)
================================================================================
  每100mL：能量204kJ, 饱脂0g, 总糖11.5g, 钠15mg, 蛋白0.2g, 纤维0g
  基线：能量0 + 饱脂0 + 总糖2(>9.0) + 钠0 = 2点
  V点：0（无FVNL成分）
  P点：0（蛋白<=1.6g）
  F点：不适用（饮料）
  最终分数：2 - 0 = 2
  星级：Cat 1 → 分数2 → 1星

================================================================================
核心设计特点
================================================================================
1. 同类对比：设计为相似产品间比较（如早餐麦片vs早餐麦片），不鼓励跨类比较
2. 2020修订：加重总糖和钠的惩罚
3. 新鲜水果蔬菜自动5星（包括部分无添加糖/盐的罐装/冷冻品）
4. <4星的产品在FSANZ法规下不能标示营养健康声称
5. 官方计算器：www.healthstarrating.gov.au/calculator

================================================================================
与Nutri-Score的主要区别
================================================================================
- HSR在同类产品间比较，Nutri-Score跨类别通用比较
- HSR有6个独立类别映射，Nutri-Score有3个类别
- HSR评估总糖，Nutri-Score重点评估添加糖
- HSR对乳制品有特殊对待（D类），Nutri-Score有红肉和奶酪子类
- HSR是自愿性的，Nutri-Score在欧洲多国是强制性的
