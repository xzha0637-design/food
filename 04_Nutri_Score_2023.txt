================================================================================
        Nutri-Score (2023修订版) — 欧盟营养标签评分系统详解
================================================================================
全称：Nutri-Score 五色营养标签
版本：2023年修订版（2023年12月31日起生效，旧产品2025年底前过渡）
采用国：法国、德国、比利时、西班牙、荷兰、卢森堡、瑞士
计算基础：每100g或每100ml
评分范围：A(最优/深绿) - E(最差/红)，底层分数为 N-P 点数差

================================================================================
核心算法
================================================================================
Nutri-Score = Σ N-points(负面) - Σ P-points(正面)

负面组分(N)：
  - 能量 (kJ) ——对脂肪/油/坚果/种子类，改用「来自饱和脂肪的能量」
  - 饱和脂肪酸 (g)
  - 糖 (g)
  - 盐 (g)——新版直接用盐而非钠
  - 甜味剂 (仅饮料，存在则+4 N点)——2023新增

正面组分(P)：
  - 蛋白质 (g)
  - 膳食纤维 (g)
  - 水果/蔬菜/豆类比例 FVP (g)——坚果和特定油(菜籽/核桃/橄榄)不再计入

┌────────┬──────────┬──────────┬──────────┐
│ P点数  │ 蛋白质(g)│ 纤维(g)  │ FVP(g)   │
├────────┼──────────┼──────────┼──────────┤
│  0     │ <=2.4    │ <=3.0    │ <=40     │
│  1     │ >2.4     │ >3.0     │ >40      │
│  2     │ >4.8     │ >4.1     │ >60      │
│  3     │ >7.2     │ >5.2     │ -        │
│  4     │ >9.6     │ >6.3     │ -        │
│  5     │ >12      │ >7.4     │ >80      │
│  6-7   │ >14->17  │ -        │ -        │
└────────┴──────────┴──────────┴──────────┘

计算方案：
  若 N >= 11 → Nutri-Score = N - 纤维点 - FVP点（蛋白质不计！）
  若 N < 11（或奶酪）→ Nutri-Score = N - 全部P点

红肉子类：蛋白质最多计2个P点（牛肉/猪/羊/马/野味及含量>=20%的产品）

类别2：脂肪、油、坚果和种子（2023新增独立类别）
  包含：植物/动物油脂、奶油、人造黄油、黄油、坚果、油籽、坚果酱(>50%含量)
  排除：栗子、椰子
  能量改用「饱和脂肪能量 = 饱脂g × 37 kJ/g」
  额外评估饱脂/总脂比率
  N>=7时蛋白质不计分；N<7时蛋白质计入

类别3：饮料（2023大幅修订，牛奶/酸奶饮料/植物饮料现归此类）
  新增甜味剂惩罚：含E420/421/953/956/964/965/966/967/968等甜味剂→+4 N点
  水为特例：自动评为A级
  能量按每100ml计算

================================================================================
核心特点
================================================================================
1. 面向消费者：五色A-E直观标签，印在包装正面
2. 三个独立类别不同计分规则：通用食品、油脂坚果种子、饮料
3. 强制阈值：A/B类不允许对儿童做广告（部分国家）
4. 官方计算工具：https://www.santepubliquefrance.fr/ 提供Excel计算器
5. 多种验证：与SU.VI.MAX、NutriNet-Santé、EPIC欧洲队列关联研究
